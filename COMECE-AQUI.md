## 📋 Checklist de Personalização

### Fase 1: Editar Conteúdo (15 minutos)
- [ ] Abrir `index.html` em editor de texto
- [ ] Mudar "Ballet Studio" para seu nome
- [ ] Atualizar seção "Sobre" com sua história
- [ ] Alterar turmas com seus horários reais
- [ ] Adicionar seu telefone e email
- [ ] Atualizar endereço (ou remover)

### Fase 2: Adicionar Mídia (20 minutos)
- [ ] Preparar suas fotos (comprimir em tinypng.com)
- [ ] Colocar fotos em pasta `/images/`
- [ ] Atualizar referências no HTML
- [ ] Adicionar vídeos (YouTube ou em `/videos/`)
- [ ] Testar se imagens aparecem localmente

### Fase 3: Testar (5 minutos)
- [ ] Rodar `python -m http.server 8000`
- [ ] Abrir `http://localhost:8000`
- [ ] Testar no navegador (Chrome, Firefox, Safari)
- [ ] Testar no celular (responsivo)
- [ ] Testar formulário de contato

### Fase 4: Publicar (10 minutos)
- [ ] Criar conta no GitHub.com
- [ ] Inicializar Git (`git init`)
- [ ] Criar repositório no GitHub
- [ ] Fazer push (`git push`)
- [ ] Ativar GitHub Pages
- [ ] Seu site está no ar! 🎉

## 🎨 Customizações Rápidas


### Alterar Logo
```html

```

### Adicionar Rede Social
```html

```
